# Quick Start — up and running in 5 minutes ⏱️

*A grown-up sets this up once; the kids run it themselves each day.*

**The app:** 👉 **[lukebriscoe.com/what-can-i-do](https://lukebriscoe.com/what-can-i-do/)**

Nothing to install, no account, no cost. It works offline once opened.

---

## 👀 Want to see it in action first?

Open the app → tap the **⚙️ cog** (top-right) → **See an example → ✨ Load example setup**.

That fills it with an example family — a planned week, some completed days, banked
money and bonus jobs — so you can click around and see how it works. When you're
ready for real, tap **⚙️ → Setup → Reset to starter setup** and follow the steps below.

---

## 🛠️ Set it up (about 5 minutes)

1. **Open the app** on the device your child will use (an iPad is ideal), then tap the
   **⚙️ cog** in the top-right. You'll land on **Setup**.

2. **Setup tab** — enter your child's **name** and pick an **avatar**. Check the
   **holiday start date** and **number of weeks**. Then set a **PIN** so the kids
   can't change the rules (you'll enter it to get back into ⚙️ next time).

3. **Rewards tab** — set the **money per completed day** (default £1), the **weekly
   limit** (default £5), and **which days can earn** (default Mon–Fri). Tweak the
   **bonus jobs** list if you like.

4. **Schedules tab** — check the **At home** schedule and edit the activities to suit
   your child. Anything ticked as **"Counts"** must be done to complete the day;
   **"Job"** marks a chore.

5. **Places tab** — use the **week planner** to set where your child is each day
   (Home, Grandparents, Summer club, On Holiday…). You can plan the whole holiday
   ahead. Each place has its own schedule.

6. **Add it to the home screen** so it opens like a real app and is more durable:
   in **Safari**, tap **Share → Add to Home Screen**.

You're done. Hand it over. 🎉

## 🧒 How the kids use it, each day

- **Today** — they see **what to do now** and tap each activity as they finish it.
  Complete the whole day and the coins drop 🪙 and the day's money is banked.
- **Bonus jobs** — extra helpful tasks they can tap to claim for a few more pennies
  (you double-check these).
- **Week** — they can see what's coming up, and tap any day to peek at the plan.
- **Bank** — their treasure: how much they've earned and saved up.

---

## 💷 Paying out & keeping it safe

- The money is a running total (an IOU). When you hand over real cash, log it in
  **⚙️ → Bank → Record a payout** so the balance stays right.
- Reward or dock money any time in **⚙️ → Bank → Manual adjustment** (put a **minus
  sign** in front to take money away for bad behaviour).
- **Back up now and then:** **⚙️ → Setup → Download a backup file**. The data lives
  only on this device, so a backup means you can restore it on a new iPad any time.

## ❓ Quick questions

- **Does refreshing or closing lose my data?** No — everything is saved on the device
  and reloads automatically.
- **Is it private?** Yes. Nothing leaves the device, there are no accounts, and every
  family's data is completely separate — even though everyone uses the same link.
- **Does it cost anything?** No. It's free and open source.
- **Do I need to fork or install anything?** No — just use the link. (Forking is only
  for developers who want to change the code or host their own copy — see the
  [README](README.md).)
